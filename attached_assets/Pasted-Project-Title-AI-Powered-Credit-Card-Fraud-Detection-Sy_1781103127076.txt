Project Title: AI-Powered Credit Card Fraud Detection System

Build a full-stack Credit Card Fraud Detection web application with modern UI and Machine Learning integration.

Technology Stack:

* Frontend: HTML, CSS, JavaScript, Bootstrap
* Backend: Python Flask/FastAPI
* Database: MySQL
* Machine Learning: Scikit-learn, XGBoost
* Authentication: JWT, OAuth
* Notifications: Email SMTP, Twilio SMS
* Charts: Chart.js
* Real-time Monitoring: WebSockets

Features Required:

* Secure login page
* User registration

* Session management
